# Django machines Module (machines.md)

## Purpose
Manages physical floor organizational divisions and storage grids: `MachineCategory`, `Machine`, `Set`, and `Rack`.

## Important Files
- [models.py](file:///backend/machines/models.py): Asset and rack layout models.
- [views.py](file:///backend/machines/views.py): Machine, Set, Category, and Rack allocation ViewSets.
